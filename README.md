# Mensa CLI

A small cloudflare worker that can be used with curl to show the current cafeteria menu.

# Usage

```sh
curl mensa.gymni.ch

curl mensa.gymni.ch/mo

curl mensa.gymni.ch/mensa-garching/fr
```
